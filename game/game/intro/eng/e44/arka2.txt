名稱：快打磚塊２ (Arkanoid 2 - Revenge of DOH，英文版)
簡介：TATIO 1988年出的第二代打磚塊遊戲，每個掉下來的英文字母分別有不
      同的功效，變化比一代多，也可以自行修改創造關卡的內容，只是不支
      援滑鼠，操控有些困難。不過這樣也比較有挑戰性些。
保護：磁片。
修改：無限多隻
      REVOFDOH.DAT
        FE 0E 97 04 74 1F
        90 90 90 90 90 90
操作：

左Shift = 打擊板左移
右Shift = 打擊板右移
Space = 發球／發射F1 = 暫停
F2 = 音樂開關
F3 = 音效開關
F9 = 編輯關卡
F10 = 結束遊戲

膠囊：

B = 跳關，直接進入下一關
C = 磁石，可將球吸停在打擊板
D = 多元球，畫面多出8個球
E = 擴充打擊板，增大一倍
I = 殘影打擊板，移動打擊板的殘影也能反彈球
L = 雷射，打擊板可發射打掉磚塊
M = 穿甲彈，球撞到磚塊仍繼續往前
N = 分裂球，球不斷一分為三，保持畫面三個球
P = 加台，增加一次陣亡機會
R = 縮小打擊板，寬度減半但分數加倍
S = 減速，球的速度變慢
T = 雙胞打擊板，打擊板旁增加一個相同的打擊板
閃動 = 特殊膠囊，有下列功能之一：
  (1)產生16個球
  (2)產生16個穿甲彈
  (3)連發雷射
