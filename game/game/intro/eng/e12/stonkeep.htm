名稱：魔石堡 (StoneKeep，英文版)
簡介：Interplay 1995年出品的角色扮演遊戲，由松崗代理發行。整個遊戲採
      直角3D視覺呈現，以團隊即時方式戰鬥，操控上則以單人為主。劇情描
      寫暗影之王襲擊魔石堡，俘虜了地球之神Thera，主角Drake長大後重回
      魔石堡，擊敗暗影之王救回Thera，並使魔石堡重回光明。遊戲一開始
      的動畫做得相當精彩，令人眼睛為之一亮，但結局動畫相較下短得太多，
      讓人有虎頭蛇尾之感。然而這個遊戲無論武器、技能、解謎、操控等等
      各部份都相當地有水準，尤其是全程大量的語音，令人很容易有身歷其
      境之感。雖然物品沒有詳細功能說明可參考，必須嚐試進行，但也不失
      其樂趣。總之，這款遊戲算是相當優秀的作品，很值得一玩。
 
保護：光碟，用DosBox玩時，應設定：
        mount d [光碟所在磁碟]:\ -t cdrom -usecd 0
      如果已燒成iso檔，可設定：
        imgmount d [iso路徑檔名] -t iso
      如此便不用放光碟，可直接安裝並進入遊戲。
檔案：sk_p12.ace = 1.2版更新檔，解壓後直接覆蓋即可。
