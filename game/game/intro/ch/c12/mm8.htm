名稱：魔法門８毀天滅地／毀滅者之日
      (Might and Magic 8 - Day of the Destroyer，繁中／簡中／英文版)
簡介：New World Computing 2000年出品的角色扮演遊戲，以3D畫面呈現，採
      即時方式戰鬥（可設定為回合制）。劇情描寫毀滅者在賈達密大陸創造
      水晶，禁錮四元素王，導致各地發生災變，元素界怪物亦失控，主角於
      是前往界間之地救出四元素王，毀滅了水晶，恢復世界的安定。本代和
      前代最大的不同是，創造人物由四人改成一人，其他隊員必須招募加入。
      由於整個隊伍可以有五人，同時有最強的飛龍傭兵，因此困難度其實並
      不算高，同時因為飛龍傭兵過強，導致其他種族傭兵顯得非常無用。不
      過整個遊戲的娛樂性仍非常好，很值得一玩。
限制：中文版必須使用相同語系之視窗系統，否則最後會無法輸入正確的中文
      字謎題答案而卡關。
保護：光碟
備註：本版為繁中版，並將愛斯卡頓的謎題答案改成可接受中英文任何一種。
備註：繁中版已更新成XP/2003可執行，更新方法為尋找繁體中文版XP補丁，執
      行後即可（已免光碟），簡體中文版亦適用，但更新後會變成繁體中文
      版（英文版不可）。英文版必須更新GrayFace Patch檔後，才能在XP系
      統執行。
備註：如果播放影音會當，可試著執行MM8Setup.Exe，將影像改成軟體模擬3D。
檔案：mm8-s.rar = 簡體中文版差異檔，覆蓋繁中版即成
      MM8patch.rar = GrayFace MM8 Patch v2.0
      mm8che.ace = 存檔修改器
      EnglishT1.rar = 繁中版裝備增強檔 (增強3～6倍)
      EnglishT2.rar = 繁中版裝備超強檔 (增強5～60倍)
      best1.rar = 單人能力裝備最強存檔
      bestall.rar = 全隊能力裝備最強存檔
修改：利用MMArchive修改EnglishT.lod裡的ITEMS.txt，可造出超強之武器。
      其他如MONSTERS.txt可調整怪物屬性，mapstats.txt可調整各地圖的重
      置週期，STDITEMS.txt可調整各級魔法物品增加屬性的範圍。
操作：（遊戲中可設定控制按鍵）

Shift+上／左右／下鍵 = 跑步／快速旋轉／快速後退
Ctrl+左右 = 側移
X = 跳躍
Enter = 即時／回合切換
PgDn = 往上看
Del = 往下看
End = 平視
PgUp = 往上飛升
Home = 往下掉落
Tab = 切換人物
S = 施展已設定法術／射箭
A = 近距攻擊／射箭
C = 施法
Q = 任務
M = 地圖
Z = 狀態
N = 記事
R = 休息
（時間不再用T鍵顯示，改成滑鼠移到右下地圖時顯示）
