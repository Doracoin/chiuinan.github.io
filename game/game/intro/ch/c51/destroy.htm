﻿名稱：滅 (中文版)
簡介：大宇1989年出品的打鬥遊戲，劇情描寫主角獨闖魔堡救出女主角。整個
      遊戲共有五個關卡，關卡間穿插了一些小遊戲，並未提供存取檔功能。
      雖然以現今眼光來看，本遊戲缺乏聲光效果，並不算是很吸引人，但因
      為這款遊戲是大宇公司自製的第一款遊戲，紀念的價值遠高於實質價值。
      喜歡懷舊的玩家也可玩看看。
保護：無
限制：單色螢幕，DosBox要設定machine=hercules。CPU速度不可過快，否則
      很快就陣亡了（cycles建議300～600左右）。
秘技：進入遊戲詢問是否要音樂時，按下Y，然後進入關卡戰鬥時，同時按下
      Ctrl、D、O、M四個鍵，成功後魔王的血會變成只有一格。
操作：

A = 拳擊
Z = 腳踢
左右鍵 = 前進／後退
上鍵 = 跳起（可配合左右鍵）
下鍵 = 蹲低
Esc = 暫停

氣功發法：先按著A再按著Z，接著按著左鍵或右鍵集氣（有集氣的動作才是正確），
放開左鍵或右鍵發出氣功，擊中敵人後才能放開A鍵與Z鍵。

攻略：（作者－青衫，攻略影片取自Chun-Yu Wang）

第1關 岩石男
特性：投擲岩石後直走。
打法：對方投擲岩石時，跳起避開，然後開始使用氣功遠距攻擊。


第2關 偽珊璞
特性：跳躍＋直衝。
打法：使用氣功攻擊，對方直衝快到時跳離。


小遊戲：猜數字，成功後恢復生命

第3關 偽冰河＋紫龍
特性：投擲盾牌。
打法：對方投擲盾牌時，跳起往前接近，蹲低用腿攻擊。


ATTE問答：答3 ATTE最帥，答Y做朋友，成功後恢復生命

第4關 女忍者
特性：投擲飛鏢、噴火、隱身、分身。
打法：對方隱身後，先準備好氣功，對方一現身即發射攻擊。最後分身時直接
      上前攻擊。


小遊戲：拼圖，成功後恢復生命

第5關之一 牛魔王
特性：直衝。
打法：使用氣功遠距攻擊，對方直衝快到時跳離。


第5關之二 火龍
特性：近距噴火。
打法：使用氣功遠距攻擊，對方接近時跳離。


結局：火龍死前會發出最後一擊，若之前有答應與ATTE做朋友，則ATTE會拼死
      擋下這一擊，否則女主角會被這一擊殺死。
